BASIC USAGE
────────────────────────────────────────────────────────────────────────────

No arguments (interactive):
  $ python ai-router.py

With subcommands (non-interactive):
  $ python ai-router.py <subcommand> [options]

Get help:
  $ python ai-router.py --help
  $ python ai-router.py <subcommand> --help

================================================================================
                          ALL COMMANDS
================================================================================

MACHINE DETECTION
  $ python ai-router.py machine detect
  Shows: detected machine ID, config path, platform, model count

CONFIGURATION
  $ python ai-router.py config show
  Shows: machine specs, performance settings, inference settings

  $ python ai-router.py validate
  Shows: configuration validation result (exit code 0 or 1)

MODELS
  $ python ai-router.py models list
  Shows: all available models with details

  $ python ai-router.py models search <keyword>
  Shows: models matching keyword (name, use case, or notes)

MODEL EXECUTION
  $ python ai-router.py run --model <model_id>
  Runs: specified model, prompts for text

  $ python ai-router.py run --model <model_id> --prompt "text"
  Runs: specified model with provided text

================================================================================
                       COMMON SCENARIOS

1. VERIFY SETUP

   $ python ai-router.py validate && echo "OK" || echo "ERROR"

================================================================================
                       SCRIPTING PATTERNS
================================================================================

BASH SCRIPT:
  #!/bin/bash
  python ai-router.py machine detect
  python ai-router.py validate
  python ai-router.py run --model qwen3-coder-30b --prompt "hello"

CONDITIONAL:
  if python ai-router.py validate; then
      python ai-router.py run --model phi4-14b --prompt "test"
  fi

LOOP:
  for model in qwen3-coder-30b phi4-14b ministral-3-14b; do
      python ai-router.py run --model $model --prompt "test"
  done

CI/CD (GitHub Actions):
  - name: Validate AI Router
    run: python ai-router.py validate
  - name: Run model
    run: python ai-router.py run --model qwen3-coder-30b --prompt "..."

================================================================================
                        EXIT CODES
================================================================================

Exit Code 0 = Success
  python ai-router.py validate   (config is valid)
  python ai-router.py machine detect  (works)
  python ai-router.py run ...    (ran successfully)

Exit Code 1 = Failure
  python ai-router.py validate   (config invalid)
  python ai-router.py run --model unknown  (model not found)

Use in scripts:
  if [ $? -eq 0 ]; then echo "Success"; else echo "Failed"; fi

================================================================================
                        TROUBLESHOOTING
================================================================================

Command not recognized:

Config validation fails:
  Check what's wrong:
  $ python ai-router.py config show
  Edit the config file to fix the issue

Model not found:
  See what models exist:
  $ python ai-router.py models list
  Or search:
  $ python ai-router.py models search <keyword>
