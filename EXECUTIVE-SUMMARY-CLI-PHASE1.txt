================================================================================
                        USAGE EXAMPLES
================================================================================

Quick Start:

  # Show help
  python ai-router.py --help

  # Check machine detection
  python ai-router.py machine detect

  # Verify configuration
  python ai-router.py validate

  # Search for coding models
  python ai-router.py models search coding

  # Run model from script
  python ai-router.py run --model qwen3-coder-30b --prompt "write a function"

Automation Example:

  #!/bin/bash
  if python ai-router.py validate; then
      python ai-router.py run --model qwen3-coder-30b --prompt "my prompt"
  else
      echo "Configuration invalid!"
      exit 1
  fi
